# Introduction

This repository is used to setup a neo-vim for the development.

## Prerequisites

| name    | version |
| ------- | ------- |
| neovim  | >=9.0   |
| Packer  | latest  |
| zip     | latest  |
| nodejs  | >=16.0  |
| python3 | 3       |

## Install

1. Copy the whole folder into `~/.config/nvim`
2. Open `nvim`, run `PackerSync` twice`

## Language Support

- Python
  - `pip install autopep8`
- Rust
- Markdown
- Lua
